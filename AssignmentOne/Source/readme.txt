Keyboard Inputs
WASD - move the model along XY plane  (one step at a time)
ZX - move the model along Z axis (one step at a time)
QE - move the model along Z axis (continuosly)
Spacebar - Reposition the model to its original coordinates

NM - Scale the model

HJ- Rotate the model along X axis
FG - Rotate the model along Y axis
VB - Rotate the model along Z axis
Note: this binding will be changed as A2 require some other binding for B key

1234 - Model Selection

Arrow keys - Change world orientation
Mouse left - Camera zoom
Mouse middle - Camera tilt
Mouse right -  Camera pan

Change the rendering Mode
P - point
L - Line
T - Triangle

Enter -  reset to original orientation
